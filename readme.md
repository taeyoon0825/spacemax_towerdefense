## 🎮 게임 개요

적들이 정해진 경로를 따라 몰려오고, 플레이어는 다양한 등급의 타워를 배치해 방어하는 **전략형 타워 디펜스 게임**입니다.  
웨이브가 진행될수록 적들의 체력과 수가 증가하며,  
각 스테이지의 5, 10단계에는 중간보스, 메인보스가 등장합니다.

---

## 🧩 주요 특징

| 기능                                 | 설명                                                                                |
| ------------------------------------ | ----------------------------------------------------------------------------------- |
| 🎯 **타워 배치 / 판매 / 업그레이드** | 클릭으로 타워 배치, 우클릭으로 판매, `U`키로 업그레이드                             |
| 💎 **타워 등급 시스템**              | `일반`, `희귀`, `영웅`, `전설`, `신화` — 등급별 공격력, 공격속도, 크기, 색상 차별화 |
| 🐝 **적(Enemy) 다양화**              | 일반 벌(`bee1`), 중간보스(`bee2`), 메인보스(`bee3`)                                 |
| 🏆 **30스테이지 클리어 목표**        | 마지막 웨이브를 클리어하면 `YOU WIN!` 화면 출력                                     |
| 🧭 **드래그 앤 드롭**                | 타워 클릭 후 마우스로 위치 재배치 가능                                              |
| 🖼️ **픽셀풍 맵 / 배경**              | 커스텀 제작된 1280×720 배경 이미지 (`map.png`) 사용                                 |
| 🧠 **자동 스테이지 관리**            | `WaveManager` 클래스가 적 스폰 및 스테이지 진행 자동 처리                           |
